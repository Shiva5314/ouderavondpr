Visual Studio Code

Visual Studio Code is een broncode-editor (een teksteditor die specifiek ontworpen is voor het bewerken van de broncode van computerprogramma's door programmeurs) ontwikkeld door Microsoft voor Windows en macOS. Het bevat ondersteuning voor debugging, ingebouwde Git-controle, syntax highlighting, intelligente code-aanvulling, codefragmenten en code refactoring. Het is ook aanpasbaar, zodat gebruikers het thema van de editor, sneltoetsen en voorkeuren kunnen wijzigen. Het is gratis en open-source, hoewel de officiÃ«le download onder een licentie valt. Visual Studio Code is gebaseerd op Electron, een framework dat wordt gebruikt om Node.js-applicaties te implementeren voor de desktop en dat draait op de Blink lay-out engine. Hoewel het gebruik maakt van het Electron-framework maakt het geen gebruik van Atom en gebruikt het in plaats daarvan dezelfde editorcomponent die wordt gebruikt in Visual Studio Team Services.
Visual Studio Code bevat ook bepaalde functies zoals syntax highlighting en dat is een optie bij sommige texteditors die tekst in verschillende kleuren en lettertypes weergeeft volgens de betekenis of categorie van bepaalde termen. Programmeertalen zoals HTML, CSS, TypeScript en Phyton. Codefragmenten zijn er ook zoals Groovy, Markdown, Nim, PHP en Swift. Intelligente codeaanvulling zijn onder andere CSS, HTML, JavaScript en TypeScript. Voor refactoring zijn er programmeertalen zoals C# en TypeScript. Bij debugging (het opsporen en verhelpen van bugs in computerprogramma's) zijn er programmeertalen als Python, JavaScript en TypeScript nodig.

Figma

Figma is een webgebaseerde app voor het bewerken van afbeeldingen en het ontwerpen van gebruikersinterfaces. Figma is te gebruiken op macOS en Windows. Je kunt het gebruiken om allerlei soorten grafisch ontwerpwerk te doen, van wireframing-websites, het ontwerpen van mobiele app-interfaces, prototyping-ontwerpen, het maken van posts op sociale media en alles daartussenin. Figma is anders dan andere grafische bewerkingstools. Vooral omdat het rechtstreeks in uw browser werkt. Dit betekent dat u toegang krijgt tot uw projecten en kunt beginnen met ontwerpen vanaf elke computer of elk platform zonder dat u meerdere licenties hoeft te kopen of software hoeft te installeren.
Figma kan je gebruiken door een desktopcomputer of laptop te hebben. Het heeft een zeer beginnersvriendelijke editor waar je helemaal zelf ontwerpen kunt maken of vooraf gemaakte sjablonen kunt gebruiken.
